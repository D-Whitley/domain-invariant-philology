## About the Recursion Explorer
Appendix B contains a React-based visualization of recursion across fractal, musical, narrative, and chemical domains. To run it locally:
1. Ensure you have [Node.js](https://nodejs.org) installed.
2. Drop the file into a React project using [Vite](https://vitejs.dev) or Create React App.
3. Install dependencies: `npm install lucide-react`.
